## 2. 투자 지표 계산 규칙

### 2.1 수익률 계산
```
일별 수익률 = (당일 종가 - 전일 종가) / 전일 종가 × 100
누적 수익률 = (현재가 - 시작가) / 시작가 × 100
연환산 수익률 = ((1 + 누적 수익률)^(365/보유일수) - 1) × 100
```

### 2.2 이동평균선 (MA)
- 단기: 5일, 20일
- 중기: 60일
- 장기: 120일, 200일
- 골든크로스: 단기 MA > 장기 MA (매수 신호)
- 데드크로스: 단기 MA < 장기 MA (매도 신호)

### 2.3 RSI (상대강도지수)
- 계산 기간: 14일 (기본값)
- 과매수: RSI > 70 → 매도 고려
- 중립: 30 ≤ RSI ≤ 70
- 과매도: RSI < 30 → 매수 고려

### 2.4 볼린저 밴드
- 중심선: 20일 이동평균
- 상단 밴드: 중심선 + 2σ
- 하단 밴드: 중심선 - 2σ
- 밴드 수축: 변동성 감소, 큰 움직임 예고

### 2.5 변동성 (Volatility)
```
일별 변동성 = 일별 수익률의 표준편차
연환산 변동성 = 일별 변동성 × √252
```

### 2.6 샤프 비율 (Sharpe Ratio)
```
샤프 비율 = (포트폴리오 수익률 - 무위험 수익률) / 변동성
무위험 수익률 기본값: 3.5% (연)
샤프 비율 > 1: 양호, > 2: 우수
```

## 3. 포트폴리오 분석 규칙

### 3.1 구성 분석
- 자산별 비중 계산 (현재가 × 보유수량 / 총자산)
- 섹터별 분산도 측정
- 상관관계 분석 (상관계수 행렬)

### 3.2 리밸런싱 신호
- 특정 자산 비중이 목표 ±5% 이상 벗어날 경우 경고 표시
- 상관관계 > 0.8인 자산 중복 보유 시 경고

### 3.3 분산 평가
- 섹터 집중도: 단일 섹터 비중 > 40% 시 경고
- 자산 집중도: 단일 종목 비중 > 30% 시 경고

### 3.4 특수 포지션 예외 처리

#### 공매도(Short Selling)
- 손익 방향이 역전됨: 가격 하락 시 수익, 상승 시 손실
- 평가손익 계산: `(매도가 - 현재가) × 수량`
- 수익률 표시: 방향 반전 경고 배지 표시 (일반 롱 포지션과 혼동 방지)
- MDD 계산: 공매도 포지션의 MDD는 가격 상승 구간에서 발생
- 인사이트 적용: 공매도 종목의 RSI 신호는 반대로 해석 (RSI >70 → 오히려 이익 신호)
- 포트폴리오 비중: 절댓값 기준으로 분산도 계산 (음수 비중 허용)

#### 레버리지(Leverage)
- 실제 시장 노출도: `포트폴리오 비중 × 레버리지 배수`
- 레버리지 2배 이상 시 자동 경고 표시
- 변동성 조정: 연환산 변동성에 레버리지 배수 곱하여 실효 변동성 표시
- MDD 조정: MDD × 레버리지 배수 = 실제 최대 손실 가능액
- 포트폴리오 집중도 경고 기준: 레버리지 반영 실효 비중으로 판단 (예: 20% 포지션 × 2배 = 40% 실효 노출)
- 샤프비율: 레버리지 사용 시 변동성 증가로 샤프비율 왜곡 가능 — 비레버리지 기준 병행 표시 권장

#### 인버스 ETF
- 벤치마크 대비 반대 수익률 추종 (-1배 또는 -2배)
- 수익률 비교 차트에서 방향 역전 명시
- RSI 신호: 공매도와 동일 — 과매수 구간이 오히려 인버스 ETF 매수 타이밍일 수 있음

## 4. 데이터 품질 규칙

- 결측값(NaN): 전일 값으로 채움 (forward fill)
- 이상치 감지: 일별 수익률 ±15% 초과 시 플래그 표시
- 최소 데이터 요건: 지표 계산에 최소 30일치 데이터 필요
- 데이터 캐싱: 동일 티커 요청 시 1시간 캐시 유지
