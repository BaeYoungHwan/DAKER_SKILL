# Skills — 시각화 선택 기준

## 1. 차트 유형 선택 기준

| 데이터 유형 | 목적 | 권장 차트 |
|-------------|------|-----------|
| OHLCV (주가) | 가격 흐름 + 거래량 | 캔들스틱 + 바 차트 |
| 단일 지표 시계열 | 추세 파악 | 라인 차트 |
| 다중 종목 비교 | 수익률 비교 | 멀티 라인 또는 바 차트 |
| 포트폴리오 구성 | 비중 시각화 | 파이 차트 또는 트리맵 |
| 상관관계 | 자산 간 관계 | 히트맵 |
| 분포 | 수익률 분포 | 히스토그램 |
| 리스크-수익 | 포트폴리오 효율성 | 산점도 |

---

## 2. 차트별 구성 규칙

### 2.1 캔들스틱 차트 (주가)
- 상승: 초록색 (#26a69a)
- 하락: 빨간색 (#ef5350)
- 거래량 바 차트를 하단에 배치 (높이 비율 3:1)
- 이동평균선 오버레이: 5일(노랑), 20일(주황), 60일(파랑)
- 볼린저 밴드: 반투명 영역으로 표시

### 2.2 수익률 비교 차트
- 기준점: 분석 시작일 = 0%
- 지수 대비 상대 수익률 함께 표시
- 색상: 종목별 고유 색상 자동 배정

### 2.3 포트폴리오 파이 차트
- 종목 수 ≤ 7: 파이 차트
- 종목 수 > 7: 트리맵
- 비중 < 3%: "기타"로 묶음

### 2.4 상관관계 히트맵
- 색상 스케일: 빨강(음의 상관) → 흰색(0) → 파랑(양의 상관)
- 대각선: 회색으로 표시
- 셀 내 수치 표시 (소수점 2자리)

### 2.5 RSI 차트
- 과매수선(70): 빨간 점선
- 과매도선(30): 초록 점선
- RSI 영역: 70 이상은 빨간 배경, 30 이하는 초록 배경

## 3. 레이아웃 규칙

### 3.1 대시보드 그리드
```
[상단] 핵심 KPI 카드 4개 (수평 배치)
[중앙 좌] 주가 캔들스틱 (너비 2/3)
[중앙 우] 포트폴리오 비중 (너비 1/3)
[하단 좌] 기술적 지표 (RSI, MACD)
[하단 우] 수익률 비교
```

### 3.2 KPI 카드 표시 항목
- 현재가 + 전일 대비 등락률
- 누적 수익률
- 연환산 변동성
- 샤프 비율

### 3.3 색상 규칙
- 양수/상승: #26a69a (초록)
- 음수/하락: #ef5350 (빨강)
- 중립/정보: #7B68EE (보라)
- 배경: #131722 (TradingView 다크)

---

## 4. 인터랙티브 기능 규칙

- 모든 차트는 줌/팬 가능
- 호버 시 상세 수치 툴팁 표시
- 기간 선택: 1M / 3M / 6M / 1Y / 3Y / 5Y 버튼
- 종목 추가/제거: 멀티셀렉트 위젯
- 차트 다운로드: PNG 버튼 제공

## 5. 반응형 기준

- Streamlit 기본 와이드 모드 사용
- 컬럼 분할은 최대 3열까지
- 모바일: 단일 컬럼으로 자동 전환 (Streamlit 기본 동작)
