# 금융 투자 대시보드 — UI/UX 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 금융 투자 대시보드 |
| 목적 | yfinance 실시간 데이터 기반 개인 투자 분석 도구 |
| 플랫폼 | Streamlit (Python) + Plotly |
| 배포 환경 | Streamlit Cloud |
| 대상 사용자 | 개인 투자자, 주식 분석에 관심 있는 일반인 |

---

## 2. 현재 구현된 기능 목록

### 2-1. 사이드바 (컨트롤 패널)

| 기능 | 설명 |
|------|------|
| 종목 검색 | 이름/티커로 검색 → selectbox로 선택 후 추가 |
| 직접 티커 입력 | 쉼표 구분 직접 입력 (Expander 내부) |
| 선택 종목 목록 | 선택된 종목 표시 + ×버튼으로 제거 |
| 분석 기간 선택 | 1개월 / 3개월 / 6개월 / 1년 / 2년 / 5년 |
| 주 분석 종목 | 멀티 종목 중 메인 분석 대상 선택 |
| 차트 옵션 | 이동평균선(MA5/20/60/120/200), 볼린저밴드, MACD 토글 |
| 벤치마크 | S&P500 / KOSPI / NASDAQ / DOW JONES / 없음 |
| 포트폴리오 비중 | 종목별 % 직접 입력 (합계 100% 유효성 검증) |

---

### 2-2. 탭 1 — 종목 분석 (`📊 종목 분석`)

| 컴포넌트 | 설명 |
|----------|------|
| 시장 지수 스트립 | 페이지 최상단 — S&P500, NASDAQ, KOSPI 등 실시간 지수 카드 |
| 대시보드 헤더 배너 | 회사명 + 티커 + 섹터/산업 + 현재가 + 전일 대비 변동률 + 라이브 점 |
| KPI 카드 (5개) | 현재가 / 누적수익률 / 연환산 변동성 / 샤프비율 / MDD |
| 재무지표 카드 (4개) | PER / PBR / EPS / 배당수익률 (데이터 있을 때만 표시) |
| 모멘텀 배지 | 1주 / 1개월 / 3개월 수익률 배지 (상승↑/하락↓ 색상 구분) |
| 52주 범위 바 | 저가~고가 트랙 위 현재 위치 점 표시 |
| 캔들스틱 차트 | OHLCV + 이동평균 오버레이 + 볼린저밴드 + 거래량 서브차트 |
| MACD 차트 | MACD 라인 / 시그널 라인 / 히스토그램 |
| RSI 섹션 | 게이지(반원형) + 라인 차트 2열 배치 |
| 최근 30일 데이터 | Expander 내부 데이터프레임 |

---

### 2-3. 탭 2 — 포트폴리오 (`💼 포트폴리오`)

| 컴포넌트 | 설명 |
|----------|------|
| 포트폴리오 KPI 스트립 | 가중 수익률 / 최고 종목 / 최저 종목 / 평균 샤프비율 |
| 혼합 통화 처리 | KRW+USD 혼합 시 환율 자동 적용 옵션 |
| 파이 차트 | 종목별 비중 시각화 |
| 종목별 성과 테이블 | 현재가 / 누적수익률 / 변동성 / 샤프 / MDD / 비중 / 베타 |
| 포트폴리오 수익률 차트 | 가중 수익률 라인 차트 (vs 벤치마크) |

---

### 2-4. 탭 3 — 비교 분석 (`📈 비교 분석`)

| 컴포넌트 | 설명 |
|----------|------|
| 수익률 비교 라인 차트 | 전 종목 누적 수익률 추이 비교 |
| 상관관계 히트맵 | 종목 간 상관계수 매트릭스 |
| 벤치마크 대비 차트 | 선택한 벤치마크와 비교 |

---

### 2-5. 탭 4 — 투자 인사이트 (`💡 투자 인사이트`)

| 컴포넌트 | 설명 |
|----------|------|
| 인사이트 카드 | 위험도(High/Medium/Info)별 색상 카드 |
| 골든크로스 탐지 | MA 골든/데드크로스 신호 표시 |
| RSI 신호 | 과매수/과매도 경고 |
| 종합 분석 요약 | 수익률, 변동성, 샤프비율 기반 종합 의견 |

## 5. 컴포넌트별 스타일 가이드 (공통)

### 5-1. 레이아웃 구조
```
┌─────────────────────────────────────────────────────┐
│  시장 지수 스트립 (Market Overview Strip)            │
├──────────────────────────────────────────────────────┤
│  [탭 네비게이션]                                     │
├──────────────────────────────────────────────────────┤
│  대시보드 헤더 배너                                  │
├───────────────┬──────────────────────────────────────┤
│               │  KPI 카드 ×5                         │
│  사이드바     │  재무지표 카드 ×4                    │
│  - 종목 설정  │  모멘텀 배지                         │
│  - 차트 옵션  │  52주 범위 바                        │
│  - 포트폴리오 │  캔들스틱 차트                       │
│    비중       │  MACD 차트                           │
│               │  RSI 섹션                            │
└───────────────┴──────────────────────────────────────┘
```

### 5-2. 카드 컴포넌트 규칙
- 모서리: `border-radius: 12px`
- 그림자: `box-shadow: 0 2px 8px rgba(0,0,0,.07)`
- 패딩: `16px 20px`
- Hover: `translateY(-2px)` + 그림자 강화

### 5-3. 색상 적용 규칙
- 상승/양수: 포인트 컬러 (초록/청록 계열)
- 하락/음수: 위험 컬러 (빨강 계열)
- 중립: 서브텍스트 컬러
- 강조: 포인트 컬러 100% / 일반 텍스트: 80% / 서브: 50%

### 5-4. 타이포그래피 계층
| 용도 | 크기 | 굵기 |
|------|------|------|
| 헤더 (회사명) | 22px | 800 |
| KPI 값 | 26px | 800 |
| 카드 레이블 | 10-11px | 700, uppercase |
| 본문 | 14px | 400 |
| 캡션/서브 | 12px | 400 |
